# Pinterest Pin Template Specifications
**For:** Canva Pro Template Creation  
**Difficulty:** Beginner  
**Time:** 30-45 minutes per template

---

## Template Overview

Create **3 Pinterest Pin Templates**:
1. Problem-Focused Pin
2. Benefit-Focused Pin  
3. Product-Focused Pin

Each template will have the same structure but different placeholder text to guide usage.

---

## Universal Specifications (All Templates)

### Dimensions
- **Width:** 1000px
- **Height:** 1500px
- **Aspect Ratio:** 2:3 (vertical)
- **Format:** Custom size in Canva

### Safe Zones
- **Margins:** 40px on all sides
- **Text Safe Area:** 60px from edges
- **Logo Placement:** Bottom-right, 40px from edges

### Layout Zones
- **Top Zone:** 0px to 300px (20% of height) - Headline area
- **Middle Zone:** 300px to 1125px (55% of height) - Image area
- **Bottom Zone:** 1125px to 1500px (25% of height) - CTA area

---

## Template 1: Problem-Focused Pin

### Step-by-Step Creation

#### Step 1: Create Design (2 mins)
1. Open Canva
2. Click **Create a design** → **Custom size**
3. Enter: Width 1000px, Height 1500px
4. Click **Create new design**

#### Step 2: Set Background (1 min)
1. Click background (or add rectangle covering entire canvas)
2. Set color to **Background Beige** (#F8F7F4)
3. Lock this layer (right-click → Lock)

**Comet Prompt:**
"Set the background color to #F8F7F4 and lock the layer"

#### Step 3: Add Safe Zone Guides (3 mins)
1. Add a rectangle (Elements → Shapes → Rectangle)
2. Set size: Width 920px, Height 1420px (1000-80px, 1500-80px)
3. Position: X 40px, Y 40px
4. Set fill to transparent, stroke to light grey (#CCCCCC), 1px
5. Lock this layer
6. This is your safe zone guide (you can hide it later)

**Comet Prompt:**
"Add a rectangle guide: 920px x 1420px, positioned at 40px, 40px, with light grey border, transparent fill"

#### Step 4: Add Top Zone (Headline Area) (5 mins)

**Add Dark Overlay:**
1. Add rectangle (Elements → Shapes → Rectangle)
2. Size: Width 1000px, Height 300px
3. Position: X 0px, Y 0px
4. Fill: Black (#000000)
5. Opacity: 50% (adjust in top toolbar)
6. Lock this layer

**Add Headline Placeholder:**
1. Add text box (Text → Add heading)
2. Type: "PROBLEM HEADLINE GOES HERE"
3. Font: Montserrat (or Poppins) - Bold
4. Size: 60px
5. Color: White (#FFFFFF)
6. Position: Center horizontally, Y: 120px (middle of top zone)
7. Alignment: Center
8. Max width: 920px (within safe zone)

**Comet Prompt:**
"Add a dark overlay rectangle: 1000px x 300px at top, 50% opacity black. Add centered headline text: 60px, bold, white, saying 'PROBLEM HEADLINE GOES HERE'"

#### Step 5: Add Middle Zone (Image Area) (3 mins)

**Add Image Placeholder:**
1. Add rectangle (Elements → Shapes → Rectangle)
2. Size: Width 1000px, Height 825px (middle zone height)
3. Position: X 0px, Y 300px
4. Fill: Light grey (#E5E5E5) with dashed border
5. Add text on top: "IMAGE GOES HERE" (small, grey, centered)
6. This is where the main image will go

**Comet Prompt:**
"Add an image placeholder rectangle: 1000px x 825px, positioned at 0px, 300px, light grey fill, with 'IMAGE GOES HERE' text"

#### Step 6: Add Bottom Zone (CTA Area) (5 mins)

**Add Dark Overlay:**
1. Add rectangle
2. Size: Width 1000px, Height 375px
3. Position: X 0px, Y 1125px
4. Fill: Black (#000000)
5. Opacity: 50%
6. Lock this layer

**Add CTA Placeholder:**
1. Add text box
2. Type: "Claim your free guide: [specific benefit]"
3. Font: Montserrat (or Poppins) - Semi-Bold
4. Size: 36px
5. Color: White (#FFFFFF)
6. Position: Center horizontally, Y: 1312px (middle of bottom zone)
7. Alignment: Center
8. Max width: 920px

**Comet Prompt:**
"Add a dark overlay rectangle: 1000px x 375px at bottom, 50% opacity black. Add centered CTA text: 36px, semi-bold, white, saying 'Claim your free guide: [specific benefit]'"

#### Step 7: Add Logo Placeholder (3 mins)
1. Add your logo (Brand Kit → Logos → Drag logo)
2. Size: 120px x 120px (maintain aspect ratio)
3. Position: X 840px (1000 - 40px margin - 120px), Y 1340px (1500 - 40px margin - 120px)
4. This is bottom-right corner

**Comet Prompt:**
"Add HappyPawsCo logo: 120px x 120px, positioned at bottom-right corner, 40px from edges"

#### Step 8: Add Template Instructions (2 mins)
1. Add small text box (top-left, outside safe zone)
2. Type: "Template: Problem-Focused Pin | Replace headline, image, and CTA"
3. Font: 12px, grey
4. This helps you remember how to use the template

#### Step 9: Save Template (1 min)
1. Click **File** → **Save**
2. Name: "Pinterest Pin - Problem-Focused Template"
3. Add to folder: "Pinterest_Pin_Templates"
4. Click **Save**

**Total Time:** ~25 minutes

## Quick Reference: Text Specifications

### Headlines
- **Font:** Montserrat or Poppins (Bold)
- **Size:** 60-72px (60px for problem/benefit, 72px for product)
- **Color:** White (#FFFFFF)
- **Position:** Top zone, centered
- **Max Width:** 920px (within safe zone)

### CTAs
- **Font:** Montserrat or Poppins (Semi-Bold)
- **Size:** 36-42px
- **Color:** White (#FFFFFF)
- **Position:** Bottom zone, centered
- **Max Width:** 920px

### Overlays
- **Color:** Black (#000000)
- **Opacity:** 50-60%
- **Position:** Behind text (send to back)

## Testing Your Templates

After creating all 3 templates:

1. **Duplicate Template 1** (right-click → Duplicate)
2. **Replace Placeholder Text:**
   - Headline: "Stuck on the M25 with a Stressed Dog?"
   - CTA: "Claim your free guide: 7 essentials for stress-free travel"
3. **Add Real Image:**
   - Upload or drag image to middle zone
   - Resize to fit (1000px x 825px)
4. **Verify:**
   - Text readable?
   - Logo visible?
   - Safe zones respected?
   - Mobile-friendly? (test on phone)

**Comet Prompt for Testing:**
"Help me replace the placeholder text with real content and add an image to test this Pinterest pin template"

---

## Common Issues & Fixes

### Issue: Text Not Readable
**Fix:** Increase overlay opacity to 60%

### Issue: Text Too Small
**Fix:** Increase headline to 72px, CTA to 42px

### Issue: Logo Too Small
**Fix:** Increase to 150px x 150px

### Issue: Can't See Safe Zones
**Fix:** Change guide color to brighter grey or red
