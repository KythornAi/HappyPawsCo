# Slide Deck Template Specifications
**For:** Canva Pro Template Creation  
**Difficulty:** Intermediate  
**Time:** 60-90 minutes for all 6 templates

---

## Template Overview

Create **6 Slide Deck Templates**:
1. Cover Slide
2. Two-Column Content Slide
3. Checklist Slide
4. Product Showcase Slide
5. Before/After Comparison Slide
6. Contact/Closing Slide

Each template follows the same design system but serves different content purposes.

---

## Universal Specifications (All Templates)

### Dimensions
- **Width:** 1920px
- **Height:** 1080px
- **Aspect Ratio:** 16:9 (standard presentation)
- **Format:** Custom size in Canva

### Safe Zones
- **Margins:** 40px on all sides
- **Text Safe Area:** 60px from edges
- **Logo Placement:** Bottom-right, 40px from edges

### Background
- **Color:** Background Beige (#F8F7F4)
- **Apply to all slides**

## Template 1: Cover Slide

### Specifications

**Layout:** Full-bleed image with text overlay

**Elements:**
1. **Background Image:** Full canvas (1920px x 1080px)
2. **Dark Overlay:** 40% opacity black, full canvas
3. **Title:** Large serif, centered, white
4. **Subtitle:** Medium sans-serif, below title, white
5. **Logo:** Bottom-right corner

### Step-by-Step Creation

#### Step 1: Create Design (2 mins)
1. Open Canva
2. Click **Create a design** → **Custom size**
3. Enter: Width 1920px, Height 1080px
4. Click **Create new design**

#### Step 2: Set Background (2 mins)
1. Add rectangle covering entire canvas
2. Set color to **Background Beige** (#F8F7F4)
3. Lock this layer

#### Step 3: Add Image Placeholder (3 mins)
1. Add rectangle: 1920px x 1080px
2. Position: X 0px, Y 0px
3. Fill: Light grey (#E5E5E5) with dashed border
4. Add text: "COVER IMAGE GOES HERE" (centered, grey)
5. This will be replaced with actual image

#### Step 4: Add Dark Overlay (2 mins)
1. Add rectangle: 1920px x 1080px
2. Position: X 0px, Y 0px
3. Fill: Black (#000000)
4. Opacity: 40%
5. Position: Above image placeholder, below text

#### Step 5: Add Title (5 mins)
1. Add text box
2. Type: "MAIN TITLE GOES HERE"
3. Font: Playfair Display (or Crimson) - Bold
4. Size: 72px
5. Color: White (#FFFFFF)
6. Position: Center horizontally, Y: 400px (vertical center)
7. Alignment: Center
8. Max width: 1800px

#### Step 6: Add Subtitle (3 mins)
1. Add text box below title
2. Type: "Subtitle or tagline goes here"
3. Font: Open Sans - Regular
4. Size: 32px
5. Color: White (#FFFFFF)
6. Position: Center horizontally, Y: 520px
7. Alignment: Center

#### Step 7: Add Logo (2 mins)
1. Add logo from Brand Kit
2. Size: 150px x 150px
3. Position: X 1730px (1920 - 40 - 150), Y 890px (1080 - 40 - 150)

#### Step 8: Save Template (1 min)
1. Name: "Slide Deck - Cover Slide Template"
2. Save to folder: "Slide_Deck_Templates"

**Total Time:** ~20 minutes

## Template 2: Two-Column Content Slide

### Specifications

**Layout:** 40% left (image), 60% right (text)

**Left Column:**
- Width: 768px (40% of 1920px)
- Height: 1080px (full height)
- Image placeholder

**Right Column:**
- Width: 1152px (60% of 1920px)
- Height: 1080px (full height)
- Background: Background Beige (#F8F7F4)
- Text content area

### Step-by-Step Creation

#### Step 1: Create Base (2 mins)
- Same as Template 1 (1920x1080px, beige background)

#### Step 2: Add Left Column Image (3 mins)
1. Add rectangle: 768px x 1080px
2. Position: X 0px, Y 0px
3. Fill: Light grey (#E5E5E5) with dashed border
4. Add text: "IMAGE GOES HERE" (centered, grey)

#### Step 3: Add Right Column Background (2 mins)
1. Add rectangle: 1152px x 1080px
2. Position: X 768px, Y 0px
3. Fill: Background Beige (#F8F7F4)
4. Lock this layer

#### Step 4: Add Title (3 mins)
1. Add text box in right column
2. Type: "SECTION TITLE GOES HERE"
3. Font: Playfair Display - Bold
4. Size: 48px
5. Color: Text Dark (#2C2C2C)
6. Position: X 808px (768 + 40px margin), Y: 80px
7. Max width: 1072px (1152 - 80px margins)

#### Step 5: Add Body Text (5 mins)
1. Add text box below title
2. Type: "Body text goes here. This is where you'll add your main content. Keep it to 6-8 lines maximum for readability."
3. Font: Open Sans - Regular
4. Size: 18px
5. Color: Text Dark (#2C2C2C)
6. Line height: 1.6
7. Position: X 808px, Y: 160px
8. Max width: 1072px

#### Step 6: Add Separator Line (2 mins)
1. Add line (Elements → Lines)
2. Width: 400px
3. Color: Accent Green (#2D5016)
4. Thickness: 2px
5. Position: X 808px, Y: 320px (below body text)

#### Step 7: Add Bullet Points (5 mins)
1. Add text box
2. Type: "• Bullet point one\n• Bullet point two\n• Bullet point three"
3. Font: Open Sans - Regular
4. Size: 18px
5. Color: Text Dark (#2C2C2C)
6. Position: X 808px, Y: 360px
7. Use green circles for bullets (Elements → Shapes → Circle, 8px, Accent Green)

#### Step 8: Add Logo (2 mins)
- Same as Template 1 (bottom-right)

#### Step 9: Save Template (1 min)
- Name: "Slide Deck - Two-Column Content Template"

**Total Time:** ~25 minutes

## Template 3: Checklist Slide

### Specifications

**Layout:** Text-only, centered checklist

**Elements:**
1. Title at top
2. Subtitle/instructions
3. Checklist items (8-10 maximum)
4. Empty checkboxes
5. Logo watermark (subtle)

### Step-by-Step Creation

#### Step 1: Create Base (2 mins)
- 1920x1080px, beige background

#### Step 2: Add Title (3 mins)
1. Add text box
2. Type: "YOUR CHECKLIST TITLE"
3. Font: Playfair Display - Bold
4. Size: 56px
5. Color: Text Dark (#2C2C2C)
6. Position: Center horizontally, Y: 120px

#### Step 3: Add Subtitle (2 mins)
1. Add text box below title
2. Type: "Before you [action], tick these off:"
3. Font: Open Sans - Regular
4. Size: 24px
5. Color: Text Dark (#2C2C2C)
6. Position: Center horizontally, Y: 220px

#### Step 4: Add Checklist Items (10 mins)
1. Create first checklist item:
   - Add square (Elements → Shapes → Square)
   - Size: 24px x 24px
   - Fill: Transparent
   - Stroke: Brown (#A87C5B), 2px
   - Position: X 200px, Y 320px
   - Add text: "Checklist item one" (Open Sans, 20px, Text Dark)
   - Position text: X 240px, Y 320px

2. Duplicate for 8-10 items:
   - Spacing: 60px between items
   - Y position increases by 60px each time

**Comet Prompt:**
"Help me create 8 checklist items with empty square checkboxes, spaced 60px apart vertically, starting at Y position 320px"

#### Step 5: Add Subtle Logo (2 mins)
1. Add logo
2. Size: 100px x 100px
3. Position: Bottom-right
4. Opacity: 20% (very subtle watermark)

#### Step 6: Save Template (1 min)
- Name: "Slide Deck - Checklist Template"

**Total Time:** ~20 minutes

## Common Issues & Fixes

### Issue: Text Overlapping
**Fix:** Increase spacing between elements, check safe zones

### Issue: Images Not Fitting
**Fix:** Use "Fit" option when placing images, maintain aspect ratio

### Issue: Colors Not Matching Brand
**Fix:** Use Brand Kit colors, verify hex codes

### Issue: Fonts Not Loading
**Fix:** Ensure fonts are added to Brand Kit, use web-safe fallbacks
