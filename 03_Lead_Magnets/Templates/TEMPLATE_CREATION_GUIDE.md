## Prerequisites: Setting Up Canva Brand Kit

### Step 1: Set Up Brand Colors (5 mins)

1. Open Canva Pro
2. Go to **Brand Kit** (left sidebar)
3. Click **Colors**
4. Add these colors:

**Primary Colors:**
- **Background Beige:** `#F8F7F4` (Name: "Background Beige")
- **Text Dark Grey:** `#2C2C2C` (Name: "Text Dark")
- **Accent Green:** `#2D5016` (Name: "Accent Green")
- **CTA Orange:** `#A87C5B` (Name: "CTA Orange")

**How to Add:**
- Click "+ Add color"
- Enter hex code
- Name the color
- Click "Add"

**Comet Prompt (if needed):**
"Help me add these 4 brand colors to Canva Brand Kit: #F8F7F4, #2C2C2C, #2D5016, #A87C5B"

---

### Step 2: Set Up Brand Fonts (5 mins)

1. In **Brand Kit**, click **Text Styles**
2. Create these text styles:

**Title Style:**
- Name: "Lead Magnet Title"
- Font: Playfair Display (or similar serif - Crimson, Merriweather)
- Size: 36-48px (we'll adjust per template)
- Weight: Bold
- Color: Text Dark (#2C2C2C)

**Body Style:**
- Name: "Lead Magnet Body"
- Font: Open Sans (or Lato, Inter)
- Size: 16-18px
- Weight: Regular
- Color: Text Dark (#2C2C2C)

**Subheading Style:**
- Name: "Lead Magnet Subheading"
- Font: Open Sans (same as body)
- Size: 24-28px
- Weight: Bold
- Color: Text Dark (#2C2C2C)

**Comet Prompt (if needed):**
"Help me create 3 text styles in Canva Brand Kit: Title (serif, bold), Body (sans-serif, regular), Subheading (sans-serif, bold)"

---

### Step 3: Upload Brand Logo (2 mins)

1. In **Brand Kit**, click **Logos**
2. Click **Upload logo**
3. Upload your HappyPawsCo logo
4. Name it "HappyPawsCo Logo"

**Comet Prompt (if needed):**
"Help me upload the HappyPawsCo logo to Canva Brand Kit"

## Template Creation Workflow

### For Each Template:

1. **Create New Design** (use exact dimensions from specs)
2. **Set Background** (use brand color)
3. **Add Safe Zones** (guides/lines for margins)
4. **Add Placeholder Elements** (text boxes, image placeholders)
5. **Style Elements** (apply brand fonts, colors)
6. **Save as Template** (in your Canva_Templates folder)

---

## Quick Reference: Template Checklist

Before saving each template, verify:

- [ ] Correct dimensions (from specs)
- [ ] Background color set (#F8F7F4)
- [ ] Safe zones marked (40px margins)
- [ ] Brand fonts applied
- [ ] Brand colors used
- [ ] Logo placeholder included
- [ ] Text hierarchy clear
- [ ] Saved with descriptive name

## Getting Help from Comet

When you need Comet's help, break tasks into small prompts:

**Good Comet Prompts:**
- "Add a rectangle shape 40px from the left edge"
- "Change this text to font size 60px"
- "Apply brand color #2C2C2C to this text"
- "Add a semi-transparent dark overlay behind this text"

**Avoid:**
- "Create the entire template" (too complex)
- "Make it look professional" (too vague)
