## Getting Started

### Step 1: Read the Guide
Start with `TEMPLATE_CREATION_GUIDE.md` - it covers:
- Setting up Canva Brand Kit
- Template creation workflow
- Getting help from Comet browser
- Quick reference checklists

### Step 2: Set Up Brand Kit
Follow the Brand Kit setup in the guide (15 mins):
- Add 4 brand colors
- Create 3 text styles
- Upload logo

### Step 3: Create Templates
Choose your starting point:
- **Pinterest Pins** (easier, faster) - 30-45 mins
- **Slide Decks** (more complex) - 60-90 mins

### Step 4: Test Templates
Create one pin and one slide using your templates to verify everything works.

## Quick Reference

### Dimensions
- **Pinterest Pins:** 1000px x 1500px (2:3 ratio)
- **Slide Decks:** 1920px x 1080px (16:9 ratio)

### Colors
- Background: #F8F7F4 (light beige)
- Text: #2C2C2C (dark grey)
- Accent: #2D5016 (dark green)
- CTA: #A87C5B (orange-brown)

### Fonts
- **Titles:** Playfair Display (serif) - Bold
- **Body:** Open Sans (sans-serif) - Regular
- **Pinterest Headlines:** Montserrat/Poppins (sans-serif) - Bold
