# Published Blogs - 2026

Final, approved blogs ready for or live on Shopify.

## What Goes Here
- Blogs you've proofread and approved
- Currently live on Shopify OR ready to publish
- Complete archive of all blog assets

## Folder Structure

Each blog maintains its complete folder structure:

```
published/2026-blogs/
└── blog-title-slug/
    ├── published.html        # Final approved HTML
    ├── image-prompts.md      # Reference for images used
    └── images/               # All blog images
        ├── image-1.png
        ├── image-2.png
        └── ...
```

## Publishing to Shopify

**Current Process (Manual):**
1. Open `published.html` in editor
2. Copy HTML content
3. Paste into Shopify blog editor
4. Upload images from `images/` folder
5. Publish on Shopify

**Future Process (Automated):**
- When we build Shopify auto-publishing
- This folder will be the source
- Blogs will auto-sync to Shopify

## Archive

This folder serves as the complete archive of:
- All published blog content
- All blog images and assets
- Image prompt references for future updates
