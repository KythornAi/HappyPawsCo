## Generation Tips

**For Google Vertex AI:**
- Copy Primary Prompt directly
- Set quality to high resolution, photorealistic style
- Generate 2-3 variations and choose the most natural-looking result
- Check that dogs look genuinely comfortable, not distressed

**For Midjourney:**
- Use Alternative Prompt (more concise)
- Add parameters: `--ar 16:9 --v 6 --style raw` for photorealism
- Use `--s 250` for moderate stylisation

**For DALL-E:**
- Use Primary Prompt, may need to shorten if exceeding character limit
- Generate in 1024x1024, then crop to 16:9
- Request "photorealistic" explicitly

## UK-Specific Details to Check

When reviewing generated images, verify:
- Car interiors look like UK vehicles (steering wheel position not visible, but general style should be European/UK)
- Any visible scenery through windows looks British (terraced houses, grey skies acceptable)
- Dogs are realistic-looking breeds common in UK
- No obviously American elements (US outlets, architectural styles)
