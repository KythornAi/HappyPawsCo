## Generation Tips

**For Vertex AI:**
- Copy Primary Prompt directly into the prompt field
- Set quality to "High resolution" and style to "Photorealistic"
- Generate 2-3 variations of each image and select the best match
- Adjust aspect ratio settings to match recommendations (16:9, 4:5, 1:1)

**For Midjourney (if using Alternative Prompts):**
- Add parameters: `--ar 16:9 --v 6 --style raw` for photorealism
- Use `--ar 4:5` for vertical images and `--ar 1:1` for square
- Add `--s 250` for moderate stylization
- Generate multiple versions using seed variations

**For DALL-E:**
- Use Primary Prompts but keep under 400 characters if prompted
- Generate at 1024x1024 then crop to desired aspect ratio
- Emphasize "photorealistic" and "natural lighting" in prompt
- Request variations if first generation doesn't match brand aesthetic
