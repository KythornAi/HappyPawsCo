# Blog Content

This folder contains blog drafts, reviewed content, and published articles.

## Structure

- **drafts/** - Raw blog content from Make.com or other sources
- **reviewed/** - Content reviewed and polished by Blog Publisher Agent
- **published/** - Final versions published to Shopify

## Workflow

1. Make.com generates blog → saves to `drafts/`
2. Blog Publisher Agent reviews → outputs to `reviewed/`
3. User publishes to Shopify → copy saved to `published/`
