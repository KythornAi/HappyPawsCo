## What Are Trust Badges?

Visual icons/text that appear on:
- Product pages (near "Add to Cart" button)
- Homepage (hero section or below featured products)
- Cart page (before checkout)
- Checkout page (payment section)

**Goal:** Reduce objections and build confidence in purchasing

## Design Recommendations

**Visual style:**
- Simple icons + short text
- Consistent colour scheme (match brand colours)
- Not too flashy or "salesy"
- Clean, modern, professional

**Size:**
- Small enough not to dominate the page
- Large enough to be noticed and read
- Mobile-friendly (stack vertically if needed)

**Layout:**
- Horizontal row of 3-4 badges on desktop
- Vertical stack on mobile
- Icon above or to the left of text


## Shopify Implementation Options

### Option 1: Manual HTML/CSS
Add badges directly to product description template in theme code.

Pros: Full control over design and placement
Cons: Need to edit theme code, not dynamic

---

### Option 2: Shopify Apps

**Trust Badge Apps:**
- "Trust Badge Master" (free)
- "Product Badges & Labels" (paid)
- "BOLD Upsell" (includes trust badges, paid)

Pros: Easy to install, no coding required, dynamic placement
Cons: Monthly cost, less control over design

---

### Option 3: Image Badges
Create badge images in Canva or similar, upload as product images or homepage sections.

Pros: Full design control, easy to implement
Cons: Not as flexible, harder to update

## IMPORTANT: Only Use Badges You Can Back Up

**DO:**
- ‚úÖ Use badges for things you actually offer (30-day returns, free delivery)
- ‚úÖ Use product-specific badges that are factually accurate (BPA-free, crash-tested)
- ‚úÖ Be honest about certifications and testing

**DON'T:**
- ‚ùå Claim certifications you don't have
- ‚ùå Use fake review counts or customer numbers
- ‚ùå Promise delivery times you can't meet
- ‚ùå Use "Best in UK" or similar claims without evidence

**Trust is everything. Only use badges you can back up with facts.**
